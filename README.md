# Análise do PIB dos municípios do Ceará (2014–2016)

Exploratory Data Analysis • Visualizações • Economia Regional

## Visão geral

Este projeto apresenta uma análise exploratória do PIB dos municípios do Ceará entre 2014 e 2016, utilizando dados oficiais do Atlas Brasil. O objetivo é transformar dados brutos em insights claros sobre o desempenho econômico das cidades, demonstrando competências em manipulação de dados, visualização e interpretação.

## Problema investigado

As diferenças de crescimento econômico entre municípios impactam planejamento público, investimento e qualidade de vida. Este estudo busca responder perguntas como:

- Quais municípios apresentaram maior crescimento de PIB no período?
- Há concentração de PIB em poucas cidades do estado?
- Como a evolução do PIB se comporta entre 2014 e 2016?
- Existem padrões regionais ou municípios que fogem da média?

## Objetivos

- Avaliar a evolução do PIB dos municípios do Ceará entre 2014 e 2016.
- Comparar o desempenho de diferentes cidades dentro do estado.
- Identificar municípios com maior e menor crescimento no período.
- Gerar gráficos informativos para apoiar a interpretação dos resultados.
- Produzir um relatório final com conclusões e possíveis interpretações.

## Tecnologias utilizadas

- Python: análise e manipulação de dados.
- Pandas: limpeza e exploração dos dados.
- Matplotlib / Seaborn: criação de gráficos.
- Jupyter Notebook: execução das análises.
- Git e GitHub: versionamento e organização do projeto.

## Estrutura do projeto

- `data/` – contém os dados originais (`pib_municipios_ce.xlsx`).
- `notebooks/` – contém o notebook utilizado para a análise (`analise_pib_ce.ipynb`).
- `reports/` – contém gráficos e relatórios gerados (`relatorio_pib_ce.pdf`).
- `.gitignore` – arquivos ignorados pelo Git.
- `requirements.txt` – lista de dependências do projeto.
- `README.md` – documentação do projeto.

## Principais pontos de interesse da análise

- Municípios que se destacam pelo maior PIB total ou maior crescimento.
- Diferenças de evolução entre municípios mais ricos e mais pobres.
- Comportamento do PIB ao longo dos anos 2014, 2015 e 2016.
- Possíveis concentrações de atividade econômica em determinadas regiões.
