# Análise do PIB dos municípios do Ceará (2014–2016)

Descrição
Este repositório reúne dados e código para analisar a evolução do PIB dos municípios do Ceará entre 2014 e 2016, a partir de dados oficiais do Atlas Brasil.​

Fonte dos dados
Os dados foram obtidos no Atlas Brasil, na seção de consulta por planilha, filtrando o estado do Ceará, todos os municípios e o indicador de PIB para o período de 2014 a 2016.​

Arquivos principais
- PIBCE.ipynb: notebook com carregamento, tratamento e análise do PIB por município.
- data.xlsx: planilha com os dados brutos baixados do Atlas Brasil.
- grafico.pdf: visualizações geradas a partir da análise.

Objetivos da análise

- Avaliar a evolução do PIB dos municípios entre 2014 e 2016.
- Comparar o desempenho de diferentes cidades dentro do estado.
